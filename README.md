Atividade de QA - FPF - 18/07/2025
Projeto de automação para testes de sanidade de login e funcionalidade de adicionar um item ao carrinho do site https://www.saucedemo.com/

Tecnologias Utilizadas:
>Cypress
>Node.js

Funções Testas:
>Login(Testes de sanidade)
  1.Login com usuário bloqueado.
  2.Login com senha inválida.
  3.Login Bem-Sucedido
>Carrinho de compras
  1.Adicionar um produto ao carrinho

Execução de testes:
npm install
npx cypress open
npx cypress run
